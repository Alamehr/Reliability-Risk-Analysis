# Reliability and Risk Analysis (Sharif University of Technology)

This repository contains my programming assignments and coursework for the graduate-level **Reliability and Risk Analysis** course at Sharif University of Technology.

Instead of traditional deterministic safety factors, this course focuses on probabilistic failure modeling. I implemented various models to calculate failure probabilities ($P_f$) and safety margins, primarily using the **Stress-Strength Interference (SSI)** framework and **Safety Index (SI)** estimations.

## Key Areas Covered:
- **Physics of Failure:** Modeling degradation, fatigue, and overstress using linear, power-law, and exponential growth curves.
- **Risk & Safety Barriers:** Quantifying probabilistic risk ($R = \langle S_i, P_i, C_i \rangle$) for active and passive safeguards.
- **Reliability Metrics:** Calculating indices like MTTF, MTBF, and availability for engineering systems.
